# Git

- Git init: Khởi tạo git repository
- Working directory/Untracked: Các file mới hoặc có thay đổi
- git add .: Thêm tất cả file vào staging
- git add <filename>: Thêm file được chọn vào staging
- Staging area/Staged: Các file đã được thêm vào git's version control, chuẩn bị commnit
- git commit -m "<message>": Thêm file vào repo/vùng commited
- Repository/Commited: Các commit
- git push origin main: Push code vào remote repo
- git status: Kiểm tra trạng thái file. Xanh: Staging/Staged, Đỏ: Working directory/untracked
